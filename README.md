A responsive web application that helps users plan road trips by adding, reordering, and visualizing destinations on a map.

## Features
- Add destinations by name
- View route on interactive map (Leaflet + OpenStreetMap)
- Calculate total distance between destinations
- Store trip data using browser localStorage
- Fully responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js
- OpenStreetMap API
